# 暗标检查工具 - 桌面应用打包说明

## 环境要求

- Node.js 18+
- Python 3.9+
- npm 或 yarn

## 打包步骤

### 1. 安装依赖

```bash
cd frontend
npm install
```

### 2. 开发模式运行（调试）

```bash
cd frontend
npm run electron:dev
```

这会同时启动：
- Vite 开发服务器 (端口 3000)
- Python 后端 (端口 8000)
- Electron 窗口

### 3. 打包为 Windows 可执行文件

```bash
cd frontend
npm run electron:build:win
```

打包完成后，可执行文件位于：
```
frontend/release/
├── 暗标检查工具 Setup 1.0.0.exe  # 安装程序
└── win-unpacked/                   # 免安装版本
```

## 目录结构

```
frontend/
├── electron/
│   └── main.js          # Electron 主进程
├── src/                 # Vue 源码
├── dist/                # Vite 打包输出
├── release/             # Electron 打包输出
└── package.json
```

## 注意事项

1. **首次打包前**：确保后端依赖已安装
   ```bash
   cd backend
   pip install -r requirements.txt
   ```

2. **Python 环境**：打包时会使用系统 Python，确保 Python 已添加到 PATH

3. **图标设置**：如需自定义图标，将 icon.ico 文件放在 `frontend/public/` 目录

4. **打包体积**：首次打包会下载 Electron 二进制文件，约 80MB

## 故障排除

### 后端启动失败
- 检查 Python 是否已安装并添加到 PATH
- 手动测试后端：`cd backend && python main.py`

### 打包失败
- 清除缓存：`rm -rf node_modules dist release`
- 重新安装：`npm install`

### 端口冲突
- 确保端口 3000 和 8000 未被占用
- 修改 `vite.config.js` 或 `backend/main.py` 中的端口配置
