# 暗标检查工具 - 启动说明

## 环境要求

- Python 3.9+
- Node.js 18+

## 方式一：桌面应用（推荐）

双击运行 `启动桌面应用.bat`，自动启动桌面应用。

或手动启动：

```bash
cd frontend
npm install
npm run electron:dev
```

## 方式二：Web 浏览器

### 1. 启动后端

```bash
cd backend
pip install -r requirements.txt
python main.py
```

后端将在 http://localhost:8000 启动

### 2. 启动前端

```bash
cd frontend
npm install
npm run dev
```

前端将在 http://localhost:3000 启动

### 3. 访问应用

打开浏览器访问 http://localhost:3000

## 方式三：打包为安装程序

```bash
cd frontend
npm run electron:build:win
```

打包完成后，安装程序位于 `frontend/release/` 目录。

## 功能说明

1. **文件上传**：拖拽或点击上传 .docx 文件
2. **格式检查**：点击"开始检查"，检查文档是否符合暗标要求
3. **一键修复**：检查完成后，点击"一键修复"自动修复格式问题
4. **下载文件**：修复完成后，可下载修复后的文件
5. **配置编辑**：点击"编辑配置"可修改暗标格式要求

## 检查项目

- ✅ 纸张大小 (A4)
- ✅ 页边距 (2.5cm)
- ✅ 页眉页脚 (禁止)
- ✅ 正文字体 (宋体 四号)
- ✅ 行间距 (固定值 28 磅)
- ✅ 段间距 (0)
- ✅ 表格字体 (仿宋 五号)
- ✅ 禁止内容 (投标人名称等)

## 注意事项

- 仅支持 .docx 格式
- 修复后的文件会保存在 backend/output 目录
- 配置文件保存在 backend/config 目录
